# Rust SNT Hornet

Rust OpenAPI client for [SNT Hornet](https://www.snt.utwente.nl/en/service/hornet).

## Installation

```shell
cargo add snt-hornet
```

## Documentation

Documentation for is available on [Docs.rs](https://docs.rs/snt-hornet/latest/snt_hornet/).
